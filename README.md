# LowMagicAge-jp

これは Low Magic Age 公式の日本語訳に、私的な追加/改変を含んだものです。  
jp フォルダ以下が翻訳の本体になります。  
en フォルダは翻訳作業中の対比と、バージョンアップ時の変更点をdiffで出しやすくする為に追加しています。
